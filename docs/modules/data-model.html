MyALM

数据模型

ALM 数据模型以账户级资产负债数据为基础，围绕现金流、期限分桶、利率曲线、情景参数和指标结果组织。

一、核心数据域

数据域说明示例字段
账户主数据资产、负债和表外业务的账户级基础信息账户号、机构、产品、币种、余额
合同属性决定现金流和重定价的合同要素起息日、到期日、利率、还款方式
现金流明细展开后的未来现金流现金流日期、现金流类型、金额、情景编号
参数表期限桶、权重、行为模型、监管参数参数类型、生效日期、参数值
利率曲线用于重定价、贴现和情景模拟曲线编号、期限、利率、币种
结果表指标计算结果批次、情景、机构、产品、指标值

二、建模原则

  • 账户级明细和汇总结果分层保存。
  • 所有结果保留批次号和情景编号。
  • 参数表保留生效日期，避免历史结果无法复现。
  • 流动性风险和利率风险共用基础数据，但结果表应按主题拆分。
