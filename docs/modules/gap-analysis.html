Gap 分析

Gap 是差额概念，通常表示某个期限桶内资产与负债的差额。必须结合上下文判断它是期限缺口还是重定价缺口。

一、基本公式

Gap = 资产金额 - 负债金额

二、期限缺口与重定价缺口

类型分桶依据主要用于关注问题
期限缺口到期日、现金流日期流动性风险未来某期限是否资金短缺
重定价缺口重定价日、利率调整日利率风险利率变化后谁先调整收益或成本

三、示例

五年固定利率贷款使用三个月存款融资时，总金额可能匹配，但三个月后负债先到期、先重定价。流动性角度看三个月期限缺口，利率风险角度看三个月重定价缺口。

四、开发关注点

  • 必须明确分桶规则。
  • 要区分到期日、现金流日和重定价日。
  • 同一账户可能在不同分析中落入不同期限桶。
